TAM ANALYSIS

市場規模 · 大幅拓寬

客群 規模 痛點對應度 付費意願
SMB IT（已是 Network AI 客戶） ~5,000 sites ⭐⭐⭐ 不會設儀表板 已在付（包套）
SMB IT（用其他管理平台 / 自架 Grafana） ~50,000 sites ⭐⭐⭐ 同樣痛點 中高（不需換平台）
MSP T1（為客戶報告數據） ~2,000 MSP ⭐⭐ 月報自動化 高（時間就是錢）
企業 IT（已有 Tableau / PowerBI 但接不到網管 data） ~10,000 企業 ⭐⭐ 需要 SQL 才能查 中（會評估）
NetOps / DevOps（觀察跨服務狀態） ~50,000 工程師 ⭐ 已有 Grafana 但設定費時 低-中（個人試用）

關鍵差異：綁在 Mode B 賣 = 只能服務第一行客群（已是客戶）。 獨立成 wedge = 可服務全部 5 個客群，TAM 至少擴大 10×。
