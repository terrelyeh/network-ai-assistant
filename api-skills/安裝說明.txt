================================================================================
  Senao API Skills Plugin — 本機安裝 / 移除操作指南
  適用工具: Claude Code
================================================================================

前置需求
--------
  - Python 3.10 或更新版本
  - Claude Code CLI（已登入）
  - Engenius Cloud 主機位址與 API Key


================================================================================
  安裝步驟
================================================================================

步驟 1：將 Marketplace 加入 Claude Code
-----------------------------------------
在 Claude Code 對話框中輸入：

    /plugins

選擇「Manage marketplaces」，再選「Add marketplace」，
輸入本 repo 在本機的路徑，例如：

    /home/your-user/senao/api-skills
    （或 Windows 路徑：C:\Users\your-user\senao\api-skills）

按下 Enter 確認後，marketplace 即加入。
此步驟只需執行一次。


步驟 2：安裝 Plugin
--------------------
在 Claude Code 對話框中輸入：

    /plugins

選擇「Install plugin」，從清單中找到「senao-api-skills」並選取。

系統會詢問安裝範圍，請依需求選擇：

    User（使用者）    — 安裝給目前帳號，所有專案皆可使用（建議）
    Project（專案）   — 僅安裝至目前開啟的 git 專案

選擇後按 Enter 確認，安裝完成。

確認安裝成功：在 /plugins 選單中選「List plugins」，
清單中應可看到 senao-api-skills。


步驟 3：確認 Python 相依套件（如遇錯誤時執行）
-------------------------------------------------
本 plugin 僅依賴 `requests` 套件（Python 標準環境通常已內建）。
若執行 API 操作時出現 ModuleNotFoundError，再手動安裝即可：

    pip install requests


步驟 4：設定 API 環境
----------------------
安裝完成後，在 Claude Code 對話框中告訴 AI 要切換的環境，例如：

    "切到 dev"
    "切到 staging"
    "切到 prod"

AI 會引導你輸入對應環境的 API Key，並自動完成環境設定，
之後即可直接執行各項 skill 操作。


步驟 5：驗證安裝
-----------------
重新啟動 Claude Code 後，在對話框中輸入：

    /skills

應可看到 senao-api-skills 相關 skill 清單，代表安裝成功。

================================================================================
  移除步驟
================================================================================

步驟 1：移除 Plugin
--------------------
在 Claude Code 對話框中輸入：

    /plugins

選擇「Uninstall plugin」，從清單中找到「senao-api-skills」並選取，
確認後即完成移除。

確認已移除：在 /plugins 選單中選「List plugins」，
確認清單中已無 senao-api-skills。


步驟 2：移除 Marketplace（選用）
---------------------------------
若不再需要使用此 marketplace，可一併移除：

在 Claude Code 對話框中輸入：

    /plugins

選擇「Manage marketplaces」，再選「Remove marketplace」，
從清單中找到對應的 marketplace 並選取，確認後移除。


步驟 3：移除 Python 套件（選用）
---------------------------------
若系統中不再有其他套件依賴 requests，可執行：

    pip uninstall requests

================================================================================
  常見問題
================================================================================

Q: 安裝後在 Claude Code 看不到 skill？
A: 請確認已重新啟動 Claude Code，plugin 在重啟後才會生效。

Q: API 呼叫回傳 401 / Unauthorized？
A: 請重新執行環境切換（例如輸入「切到 dev」），確認 API Key 正確。

Q: 想只在單一專案使用，不影響其他專案？
A: 在 /plugins > Install plugin 安裝時，當系統詢問安裝範圍，
   選擇「Project」即可。
